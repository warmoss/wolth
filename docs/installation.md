# Installation

## Stable release

To install wolth, run this command in your terminal:

```sh
uv add wolth
```

Or if you prefer to use `pip`:

```sh
pip install wolth
```

## From source

The source files for wolth can be downloaded from the [Github repo](https://github.com/warmoss/wolth).

You can either clone the public repository:

```sh
git clone https://github.com/warmoss/wolth
```

Or download the [tarball](https://github.com/warmoss/wolth/tarball/main):

```sh
curl -OJL https://github.com/warmoss/wolth/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd wolth
uv sync
```
