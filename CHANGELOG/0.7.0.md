# 0.7.0 (2026-06-25)

## Highlights

This release marks a significant expansion of `wolth` from a basic scaffold into a practical Python toolkit. Three new modules have been added — **collections**, **lang**, and **util** — providing enhanced data structures, a simple counter utility, and general-purpose file / JSON / process helpers.

## Features

### collections — Enhanced Data Structures

- **`EnhancedDict`** — A `dict` subclass supporting attribute-style access (`d.key`), auto-enhancement of nested containers, and recursive conversion back to plain `dict`. ([`src/wolth/collections/enhanced/enhanced_dict.py`](src/wolth/collections/enhanced/enhanced_dict.py))
- **`EnhancedList`** — A `list` subclass that automatically wraps nested plain containers into enhanced types on insertion / retrieval. ([`src/wolth/collections/enhanced/enhanced_list.py`](src/wolth/collections/enhanced/enhanced_list.py))
- **`converter`** — Recursive conversion helpers (`enhance` / `to_dict`) for transparently moving between plain and enhanced collection types. ([`src/wolth/collections/enhanced/converter.py`](src/wolth/collections/enhanced/converter.py))

### lang — Language-Level Utilities

- **`Counter`** — A simple, thread-unsafe incrementing counter with `get()`, `incr()`, `incr_and_get()`, and `get_and_incr()` semantics. Useful for ID generation and iteration tracking. ([`src/wolth/lang/counter.py`](src/wolth/lang/counter.py))

### util — General-Purpose Utilities

- **`util.files`** — File and directory I/O helpers covering read / write, path joining, existence checks, recursive directory create / delete, file removal, and ZIP extraction. ([`src/wolth/util/files.py`](src/wolth/util/files.py))
- **`util.jsons`** — JSON serialization / deserialization with automatic `EnhancedDict` conversion and a custom `ComplexEncoder` that handles `date` and `datetime` objects. ([`src/wolth/util/jsons.py`](src/wolth/util/jsons.py))
- **`util.processes`** — Process-tree management with a graceful-then-forceful kill strategy (`kill` / `kill_process_tree`), preventing orphan processes. ([`src/wolth/util/processes.py`](src/wolth/util/processes.py))
